# Vatsal Vadia — Growth Operations Portfolio

A premium, high-converting personal portfolio for **Vatsal Vadia**, a Growth Operations & Performance Marketing Expert with 9+ years of experience.

Built with **React + Vite**. Optimized for SEO, AEO (Answer Engine Optimization), and GEO (Generative Engine Optimization).

---

## 🚀 Tech Stack

- **React 19** — Component-based UI
- **Vite 8** — Lightning-fast build tool
- **Lucide React** — Icon library
- **Vanilla CSS** — No external CSS frameworks

---

## 📦 Getting Started (Local)

```bash
# Install dependencies
npm install

# Start local dev server
npm run dev

# Build for production
npm run build

# Preview the production build
npm run preview
```

---

## ☁️ Deploy to Vercel

This project is pre-configured for Vercel via `vercel.json`.

### Option 1: Deploy via GitHub (Recommended)
1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → **Add New Project**
3. Import your GitHub repo `vatsalvadia/Portfolio`
4. Vercel will auto-detect the framework as **Vite**
5. Click **Deploy** ✅

> No manual configuration required — `vercel.json` handles everything.

### Option 2: Deploy via Vercel CLI
```bash
npm i -g vercel
vercel
```
